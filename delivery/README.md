# 交付与验收材料

这个分类收纳用于软件项目交付、验收、测试报告、用户手册和运维手册编写的 skills。

| Skill | 目录 | 用途 |
| --- | --- | --- |
| 交付材料整理 | [delivery-materials/](delivery-materials/) | 根据项目范围、Git 提交、接口变更、页面功能、截图或验收说明，生成正式中文交付材料。 |

## 适用场景

- 项目上线前需要补齐交付文档。
- 根据一批 Git 提交或功能清单整理验收材料。
- 为 Web 功能或开放接口编写用户使用手册。
- 编写功能测试、性能测试、测试验收报告和运维手册。
- 已有截图后，把截图嵌入 Word 交付文档。
